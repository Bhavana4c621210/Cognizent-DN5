# Module 4 – NUnit and Moq

## Objective

This module demonstrates the implementation of unit testing using the NUnit framework and mocking dependencies using the Moq framework in C#.

## Exercises Included

### 1. NUnit Hands-on

#### Description
- Created an NUnit Test Project.
- Added a project reference to the Class Library.
- Implemented unit tests using NUnit.
- Executed and verified test cases successfully.

#### Concepts Covered
- Unit Testing
- NUnit Framework
- Assertions
- Test Fixtures
- Test Cases

#### Description

- Understand Unit Testing using NUnit.
