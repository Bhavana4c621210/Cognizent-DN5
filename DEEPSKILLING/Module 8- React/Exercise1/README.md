# ReactJS Lab 1 - My First React Application

## Objective
Create a React application named **myfirstreact** and display the heading:

> Welcome the first session of React

## Prerequisites
- Node.js
- npm
- Visual Studio Code

## Steps Performed
1. Installed Node.js and npm
2. Created React project using:
   ```bash
   npx create-react-app myfirstreact
   ```
3. Modified `src/App.js`
4. Ran the application using:
   ```bash
   npm start
   ```
5. Verified the output in the browser.

## Output

The application displays:

```
Welcome the first session of React
```

## Technologies Used

- ReactJS
- JavaScript
- HTML
- CSS
- Node.js
